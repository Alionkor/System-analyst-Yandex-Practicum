# User Story Map, MVP, прототипы и DFD

Проект по анализу требований и подготовке аналитических артефактов для приложения в рамках Agile-подхода.

## Роль

Системный аналитик.

## Выполненные задачи

* изучил интервью с заинтересованными лицами;
* выделил потребности пользователей и ключевые требования к приложению;
* участвовал в формировании карты пользовательских историй;
* описал пользовательские истории;
* выделил MVP для первого релиза;
* подготовил прототипы экранов;
* составил контекстную DFD-диаграмму;
* составил логическую DFD-диаграмму;
* участвовал в командной проверке и согласовании аналитических артефактов.

## Артефакты

* карта пользовательских историй;
* MVP;
* пользовательские истории;
* прототипы экранов;
* контекстная DFD;
* логическая DFD.

## Инструменты и нотации

* User Story Mapping;
* MVP;
* User Story;
* DFD;
* Figma;
* Miro;
* diagrams.net.

## Результат

Подготовлен комплект аналитических артефактов для проектирования первого релиза приложения. Материалы могут быть использованы командой разработки для уточнения объёма MVP, проектирования интерфейсов и описания потоков данных.
