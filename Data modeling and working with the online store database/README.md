# Логическая модель данных и словарь данных

Проект по доработке спецификации требований к веб-приложению.

## Роль

Системный аналитик.

## Выполненные задачи

* проанализировал существующую спецификацию требований;
* составил логическую модель данных;
* оформил ER-диаграмму в нотации Crow’s Foot;
* привёл модель данных к третьей нормальной форме;
* описал сущности, связи и ключевые атрибуты;
* подготовил словарь данных;
* добавил модель данных и словарь в обновлённую спецификацию.

## Артефакты

* спецификация требований;
* логическая модель данных;
* ER-диаграмма;
* словарь данных.

## Инструменты и нотации

* ER-моделирование;
* Crow’s Foot;
* 3НФ;
* Google Docs;
* diagrams.net.

## Результат

Подготовлена обновлённая версия спецификации требований, дополненная логической моделью данных и словарём данных. Материалы могут быть использованы командой разработки для проектирования структуры базы данных и уточнения требований к данным приложения.
