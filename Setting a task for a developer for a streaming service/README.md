# Анализ REST API и постановка задачи на доработку

Проект по анализу требований, изучению существующего REST API и подготовке задачи на доработку API для сервиса.

## Роль

Системный аналитик.

## Выполненные задачи

* изучил интервью с представителем заказчика;
* выделил требования к новой функции сервиса;
* проанализировал существующую архитектуру и API;
* протестировал доступные операции API через Swagger;
* составил диаграмму последовательности для новой функции;
* описал сценарий взаимодействия между системами;
* подготовил задачу на разработку новой операции API.

## Результат

Подготовлены материалы для доработки API: проанализированы требования заказчика, проверены существующие операции, описан сценарий взаимодействия и сформулирована задача для команды разработки. Материалы могут быть использованы для реализации новой функции сервиса и согласования изменений в API.
