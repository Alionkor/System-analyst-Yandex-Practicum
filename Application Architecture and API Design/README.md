# Application Architecture and API Design

A project focused on analyzing the application architecture, updating the C4 model, and designing a REST API for new functionality.

## Role

System Analyst.

## Completed Tasks

* analyzed the transcript of a meeting with the architect and developer;
* updated the C4 container diagram;
* clarified the relationships between application services;
* identified digital capabilities for the new functionality;
* formulated Job Stories;
* defined user actions and scenario steps;
* designed the API profile for the new functionality;
* described API operations;
* prepared YAML API documentation using Swagger / OpenAPI.
