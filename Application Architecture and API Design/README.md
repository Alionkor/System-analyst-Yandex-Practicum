# Архитектура приложения и проектирование API

Проект по анализу архитектуры приложения, дополнению C4-модели и проектированию REST API для новой функциональности.

## Роль

Системный аналитик.

## Выполненные задачи

* проанализирована расшифровка встречи с архитектором и разработчиком;
* дополнена C4-диаграмма контейнеров;
* уточнены связи между сервисами приложения;
* определены цифровые возможности новой функциональности;
* сформулированы Job Stories;
* определены действия пользователей и шаги сценариев;
* спроектирован профиль API для новой функциональности;
* описаны операции API;
* подготовлена YAML-документация API в Swagger / OpenAPI.
