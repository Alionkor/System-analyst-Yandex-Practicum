# Аналитика и прототипирование приложения для умного дома

Итоговый проект по анализу требований, моделированию процессов и данных, подготовке прототипа и документации для первой версии приложения умного дома.

## Роль

Системный аналитик.

## Выполненные задачи

* проанализировал интервью с заинтересованными лицами;
* выделил потребности пользователей и требования к первой версии приложения;
* разработал карту пользовательских историй;
* определил состав MVP;
* описал пользовательские истории;
* подготовил критерии и сценарии приёмки в формате Gherkin;
* разработал DFD контекстного уровня;
* разработал DFD логического уровня;
* подготовил логическую ER-диаграмму в нотации Crow’s Foot;
* выполнил нормализацию модели данных до 3НФ;
* разработал словарь данных;
* подготовил диаграмму структуры интерфейсов;
* разработал прототип приложения в Figma;
* подготовил программу и методику испытаний;
* оформил презентацию результатов проекта.

# Analytics and Prototyping for a Smart Home Application

A final project focused on requirements analysis, process and data modeling, prototype preparation, and documentation for the first release of a smart home application.

## Role

System Analyst.

## Completed Tasks

* analyzed interviews with stakeholders;
* identified user needs and requirements for the first application release;
* created a user story map;
* defined the MVP scope;
* described user stories;
* prepared acceptance criteria and acceptance scenarios in Gherkin format;
* created a context-level DFD;
* created a logical-level DFD;
* prepared a logical ER diagram using Crow’s Foot notation;
* normalized the data model to Third Normal Form;
* prepared a data dictionary;
* created an interface structure diagram;
* developed an application prototype in Figma;
* prepared a test plan and test procedure document;
* prepared a project presentation.
